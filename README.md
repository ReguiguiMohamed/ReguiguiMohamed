I build scalable data systems while wrestling with bigger questions — about power, memory, inequality, and the thin line between order and collapse.

As a computer science student focused on data engineering, I design pipelines that turn raw noise into usable signal. Yet I know most of what we call “data” is just another story we tell ourselves. Some days the code flows with ruthless clarity. Other days the mind spirals — classic ADHD wiring meeting a Nietzschean suspicion of all grand systems.

I read Kafka when debugging feels too much like *The Trial*, and I return to code because building resilient infrastructure is one of the few honest ways left to impose meaning on entropy.

My repositories are scattered reflections of these obsessions. Feel free to wander through them.
